
$0000...$03FF - память RAM - оперативная память - 1 Kb

    $01FF...$03FF - область памяти RAM в которой дополнительно может быть организован стек

$4000...$427F - память EEPROM - энергонезависимая память - 640 b

$4800...$483F - OPTION Registers - энергонезависимые конфигурационные данные - 64 b

$4865...$4870 - Unique ID - доступная только для чтения область памяти содержащая информацию уникальную для любого экземпляра STM8

$5000...$540F - HW Registers - регистры ввода/вывода - 1 Kb

$7F60...$7F60 - Global configuration register и Interrupt Software priority registers

$8000...$9FFF - память FLASH - энергонезависимая память программ - 8 Kb

    $8000...$807F - область памяти FLASH содержащая таблицу векторов прерывания

    поэтому программа начиается с $8080
